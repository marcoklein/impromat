Line Gag-Props

By learnimprov | May 5, 2019
0 Comment

Synonyms

Most Uses Of.

Introduction

The performers will make a series of puns, gags and actions using real living prop. The joke cannot use the object as what it is.

Description

Line Gag is a structure that is both an exercise and a handle. This is the handle version of the structure called Props. For a detailed explanation of line gags see the exercise Line Gags

For Props the host grabs some object that is unfamiliar to the performers and throws it on the stage. The performers step forward grab the object and do stupid things with it. The object should not be identified as what it actually is. For example a rolling pin could never be used as a rolling pin. The goal of the handle is to come up with as many things as the object is not. (probably my favourite sentence)

If the host found a barbecue, like the one above, the performers would step forward and do stupid none barbecue things with it. For example, take the lid off and make it your hat, pretend it is your robot companion, turn it into a toilet, etc, etc.

Gimmicks

  • It’s a phone. It’s grenade. It’s a surf board. It’s a turd. It’s a toilet. It’s toilet surf board phone grenade.

  • Two Hander – two performers pass the prop back and fort.
